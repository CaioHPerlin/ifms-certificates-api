# IFMS Certificates API

## Description

The IFMS Certificates API is a backend application designed to help students of the IFMS TADS (Análise e Desenvolvimento de Sistemas) course manage their extracurricular course certificates. The primary goal of this API is to allow users to register their certificates and calculate the remaining hours needed to meet the course requirements based on specific rules.

> **Note:** This project is currently under development, and the described functionality is not yet implemented. There are no releases available at this time.

## Project Setup

To set up the project, ensure you have [pnpm](https://pnpm.io/) installed, then run:

```bash
pnpm install
```

## Running the Application

You can run the application in different modes:

- **Development mode:**
  ```bash
  pnpm run start
  ```

- **Watch mode:**
  ```bash
  pnpm run start:dev
  ```

- **Production mode:**
  ```bash
  pnpm run start:prod
  ```

## Testing

This project includes a testing setup. You can run the following commands:

- **Unit tests:**
  ```bash
  pnpm run test
  ```

- **End-to-end tests:**
  ```bash
  pnpm run test:e2e
  ```

- **Test coverage:**
  ```bash
  pnpm run test:cov
  ```
